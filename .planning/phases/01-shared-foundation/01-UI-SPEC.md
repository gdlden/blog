# Phase 1 — UI Design Contract

> Visual and interaction contract for the shared foundation phase (auth shell, navigation, logout). Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Vue 3 + Tailwind CSS 4) |
| Preset | not applicable |
| Component library | native Vue + Tailwind utilities |
| Icon library | none for this phase (text labels or simple SVGs preferred) |
| Font | system sans-serif stack (`font-sans`) — default Tailwind stack |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, nav item gaps |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, form group gaps |
| xl | 32px | Layout gaps, page side margins |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: none

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 500 | 1.4 |
| Heading | 20px | 600 | 1.2 |
| Display | 28px | 600 | 1.2 |

Notes:
- Use `font-sans` (system sans stack) for all text.
- Keep weights to 400 (regular) and 600 (semibold) only.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#ffffff` / `bg-white` | Page background, main surfaces |
| Secondary (30%) | `#f3f4f6` / `bg-gray-100` | Sidebar, nav container, card backgrounds |
| Accent (10%) | `#2563eb` / `bg-blue-600` | Active nav item, primary buttons, focused input rings |
| Destructive | `#dc2626` / `bg-red-600` | Logout action only |

Accent reserved for:
- Active navigation item background/text
- Primary action buttons (e.g., login submit)
- Focus rings on interactive inputs (`focus:ring-blue-600`)

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | 登录 (Log in) |
| Empty state heading | 暂无内容 |
| Empty state body | 当前页面没有可显示的内容。 |
| Error state | 登录失败，请检查用户名或密码后重试。 |
| Destructive confirmation | 退出登录: 确定要退出当前账号吗？ |

Notes:
- The login page currently uses Chinese labels (`用户名`, `密码`, `登录`, `重置`). Preserve this for consistency.
- Navigation labels for the shared shell:
  - 博文 (Blog)
  - 债务 (Debt)
  - 退出登录 (Log out)
