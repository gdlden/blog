  <action>
    Create the debt Pinia store at price_recorder_vue/src/stores/debtStore.ts following the blogStore.ts pattern, with additional summary computed properties.

    Use Setup Store pattern (Composition API style):

    State (refs):
    - debts: ref<Debt[]>([])
    - currentDebt: ref<Debt | null>(null)
    - loading: ref(false)
    - error: ref<string | null>(null)
    - total: ref(0)
    - currentPage: ref(1)
    - pageSize: ref(10)

    Getters (computed):
    - debtCount: computed(() => debts.value.length)
    - totalPages: computed(() => Math.ceil(total.value / pageSize.value))
    - totalDebt: computed(() => {
        return debts.value.reduce((sum, d) => sum + (parseFloat(d.amount) || 0), 0)
      })
    - repaidAmount: computed(() => {
        // Calculate based on status or fee/tenor logic
        return debts.value
          .filter(d => d.status === '已结清' || d.status === 'repaid')
          .reduce((sum, d) => sum + (parseFloat(d.amount) || 0), 0)
      })
    - outstandingAmount: computed(() => {
        return debts.value
          .filter(d => d.status !== '已结清' && d.status !== 'repaid')
          .reduce((sum, d) => sum + (parseFloat(d.amount) || 0), 0)
      })

    Actions (async functions):
    1. fetchDebts(page?: number, size?: number): Promise<void>
       - Set loading=true, error=null
       - Call debtApi.getDebts(String(page), String(size))
       - Update debts, total, currentPage, pageSize
       - Set loading=false
       - On error: set error message

    2. createDebt(data: Omit<Debt, 'id'>): Promise<void>
       - Set loading=true
       - Call debtApi.createDebt(data)
       - Call fetchDebts() to refresh list
       - Set loading=false
       - On error: throw error for component to handle

    3. updateDebt(data: Debt): Promise<void>
       - Set loading=true
       - Call debtApi.updateDebt(data)
       - Call fetchDebts() to refresh list
       - Set loading=false
       - On error: throw error for component to handle

    4. deleteDebt(id: string): Promise<void>
       - Set loading=true
       - Call debtApi.deleteDebt(id)
       - Filter debts array to remove deleted debt
       - Set loading=false
       - On error: throw error for component to handle

    5. setCurrentDebt(debt: Debt | null): void
       - Set currentDebt.value = debt

    Return object with all state, getters, and actions.

    Per D-08 through D-10: Pinia store manages list data, current detail, loading states, errors.
    Per DEBT-06: Summary getters calculate total, repaid, and outstanding amounts.
