    - `github.com/go-kratos/kratos/v2/transport/http`
    - `github.com/go-kratos/kratos/v2/transport/http/httputil`

    Define the struct exactly as:
    ```go
    type UnifiedResponse struct {
        Code    int         `json:"code"`
        Message string      `json:"message"`
        Data    interface{} `json:"data,omitempty"`
    }
    ```

    Implement `CustomResponseEncoder(w stdhttp.ResponseWriter, r *stdhttp.Request, v interface{}) error`:
    - If `v == nil`, set `Content-Type: application/json`, write header 200, and encode `UnifiedResponse{Code: 200, Message: "success"}`.
    - If `v` implements `http.Redirector`, call `url, code := v.(http.Redirector).Redirect()` and perform `stdhttp.Redirect(w, r, url, code)`, then return `nil`.
    - Otherwise, get codec via `http.CodecForRequest(r, "Accept")`, set `Content-Type` via `httputil.ContentType(codec.Name())`, and return `codec.Marshal(w, UnifiedResponse{Code: 200, Message: "success", Data: v})`.

    Implement `CustomErrorEncoder(w stdhttp.ResponseWriter, r *stdhttp.Request, err error)`:
    - Convert error with `se := errors.FromError(err)`.
    - Compute business code via `businessCode := mapHTTPToBusinessCode(int(se.Code))`.
    - Get codec via `http.CodecForRequest(r, "Accept")`.
    - Marshal `UnifiedResponse{Code: businessCode, Message: se.Message}`.
    - Set `Content-Type` via `httputil.ContentType(codec.Name())`.
    - Compute HTTP status: `httpStatus := int(se.Code)`; if `< 100 || > 599`, set `httpStatus = 500`.
    - Write header with `httpStatus`, then write body.

    Implement `mapHTTPToBusinessCode(httpCode int) int` with this exact mapping:
    - `400` -> `1001`
    - `401` -> `1002`
    - `404` -> `1003`
    - `500` -> `1004`
    - default -> `httpCode`
